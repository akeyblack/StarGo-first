Get started

This API provides programmatic access to speach recognition service and yelp.com parsing

Supported formats for speach recognition:
aac, avi, flac, webm, mp4, mp2, mp3, mjpg, ogg and many others... Try yourself!

upload media to server

To upload media you need to make a POST call WITH BODY object to the following url:
https://stargo-first.herokuapp.com/contents/upload


Body parameters

Field Type Description
file File Your media file

get transcription of your uploaded media file

To get transcription of your file you need to make a GET call to the following url:
https://stargo-first.herokuapp.com/contents/*your_file_name*


Example

https://stargo-first.herokuapp.com/contents/itsmyspeach.mp3

Also you can add email notifications by adding "email" query. Example:

https://stargo-first.herokuapp.com/contents/itsmyspeach.mp3?email=myemail@gmail.com

Important

You must upload your file before it's transcription and filenames must be identical in both calls
If transcription takes too long, service will send message "Transcription in process". Just wait and send call again.

upload and get list of places from yelp.com by city

To upload the list you need to make a GET call to the following url:
https://stargo-first.herokuapp.com/places/upload/*city*


Example

https://stargo-first.herokuapp.com/places/upload/Chicago

Also you can add email notifications by adding "email" query. Example:

https://stargo-first.herokuapp.com/places/upload/Chicago?email=myemail@gmail.com

get uploaded places by filter

To get places data you need to make a GET call to the following url with query:
https://stargo-first.herokuapp.com/places?hour=Day_time&city=City&amenities=amenity1,amenity2

Important

All queries are optional, amenities.length <=2


Examples

https://stargo-first.herokuapp.com/places?hour=Monday_13&city=Detroit&amenities=Offers Takeout,Accepts Credit Cards

https://stargo-first.herokuapp.com/places?hour=Monday_13

https://stargo-first.herokuapp.com/places?city=Detroit&amenities=Offers Takeout

https://stargo-first.herokuapp.com/places

Get and patch reservations

To get reservations in process you need to make a GET call to the following url:
https://stargo-first.herokuapp.com/reservations

To approve or deny reservations you need to make a PATCH call to the following url with body { approve: true/false }:
https://stargo-first.herokuapp.com/reservations/*reservation_id*

Errors

This API uses the following standart error statuses:

Error Status Meaning
400 Your file is corrupted or has unsupported format
404 We can't find file with given filename
50X Sorry, looks like an error occurred on server side, try again later
